### prisma generate requires DATABASE_URL to be set
- **Where**: `prisma.config.ts:14`
- **Pattern**: `prisma.config.ts` calls `env("DATABASE_URL")`. The resolver throws on missing/empty even for schema-only commands like `prisma generate`. Any non-empty string satisfies it; no real connection happens during generate.
- **Example**: `DATABASE_URL='postgresql://prisma-generate-stub:stub@localhost:5432/stub' npx prisma generate`

### Verifier script shape (DB-free flavour)

- **Pattern**: `import "dotenv/config"`, local `assert(cond, msg)` that `console.error`s and `process.exit(1)`, single `async function main()` invoked at module bottom with `.catch(...)`. For pure-function verifiers, drop the `db` import / fixtures / cleanup.
